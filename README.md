# Hệ thống Quản lý Công ty (QLNS)

Hệ thống quản lý toàn diện cho công ty bao gồm:
- Quản lý khách hàng
- Quản lý doanh nghiệp
- Quản lý công việc nội bộ
- Quản lý thu chi

## Cài đặt

```bash
npm install
```

## Chạy ứng dụng

```bash
npm start
```

Ứng dụng sẽ chạy tại http://localhost:3000

## Cấu trúc dự án

- `src/modules/` - Các module chính của ứng dụng
- `src/mockapi/` - Mock API và dữ liệu giả
- `src/routes/` - Cấu hình routing
- `src/components/` - Các component dùng chung
